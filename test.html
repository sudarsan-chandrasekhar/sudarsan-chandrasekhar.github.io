Project: TwitchTV JSON API for Free Code Camp

My attempt at the Twitch project for FCC.

This app uses the Twitch API to check if certain channels are currently streaming, and will display the current details of the stream if it is online. If the channel/stream does not exist or was closed by the streamer, then 'Channel does not exist' will display.

Technologies Used:

  • Bootstrap
  • jQuery
  • Twitchtv API

Future Features:

  • Implement a form that allows users to input their own set of Twitch streams
